instructions

  1. Set the timer, which should be on "work" and is defaulted to twenty-five minutes. The tomato should ripen as you focus on your task.
  2. Stop working when it ends and the tomato drops.
  3. If the number of work intervals, or "pomodoros", is less than four so far, then take a short rest. For this purpose, the timer is three minutes by default. The tomato should grow back! Now back to the first step.
  4. If the number of work intervals has gone back down to 0, then take a long rest. The tomato will grow back slowly! For this purpose, the timer is fifteen minutes by default. Now back to the first step, or pick something else to do.
  5. The "start/stop" buttons and the "work/break" buttons can be toggled and will also automatically change when the timer is up. The time can be manually changed when paused.
  6. The "pomodoros" counts the number of completed work intervals so far, and will reset to zero on the fourth interval. The current interval can be manually changed when paused.
